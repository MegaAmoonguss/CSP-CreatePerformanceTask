# GMath
The old GMath Java library reworked and greatly improved. Some important functions still have to be added.

## Install
To install, download the source and run:
```
$ pip install .
```
Pass `--upgrade` to upgrade.

## Usage
To get started, run:
```
$ gmath --help
```
Usage example:
```
$ gmath fraction 0.08(3)
0.08(3) = 1/12
```
